1- What is this?
	* This is a J!Gov installation/upgrade package to version 3.x
	* J!Gov Official site: http://www.jgov.net
	* Detailed changes in the Changelog: https://github.com/site4com/j-gov-3

2- What is J!Gov?
	* J!Gov is a Content Management System (CMS) which enables you to build accessible Web sites and powerful online applications.
	* It's a free and OpenSource software, distributed under the GNU General Public License version 2 or later
	* This is a simple and powerful web server application and it requires a server with PHP and either MySQL, PostgreSQL, or SQL Server to run it.
	More details here: http://www.jgov.net/en/JGov/what-is-jgov.html

3- Is J!Gov for you?
	* J!Gov is the right solution for any accessible content web project: see Features - http://www.jgov.net/en/JGov/jgov-features.html
	* Try out our online demo: http://site4com.net/jgov3/

4- How to find a J!Gov translation?
	* Repository of accredited language packs: http://www.jgov.net/en/download-jgov.html
	
5- Learn J!Gov
	* Read Getting Started with J!Gov to find out the basics: https://docs.joomla.org/J3.x:Getting_Started_with_Joomla!
	
6- What are the limits of J!Gov?
	* J!Gov sites can be extended in functionalities with Extensions that you can create (or download) to suite your needs.
	* There are lots of ready made extensions that you can download and install.
	* See the J!Gov Extensions Directory (J!GOV-ED): http://www.jgov.net/en/cat_parent.html

7- Is it easy to change the layout display?
	* The layout is controlled by templates that you can edit.
	* There are lots of ready made templates that you can download.

8- Ready to install J!Gov?
	* See minimum requirements here: http://www.joomla.org/technical-requirements.html
	* How do you install J!Gov ? - https://docs.joomla.org/J3.x:Installing_Joomla
	* Start your J!Gov experience building your site with a local test server.
	When ready it can be moved to an on-line hosting account of your choice.
	See the tutorial: https://docs.joomla.org/Installing_Joomla_locally
